# SPARK Patch Notes

Added - Global Bot/User Ignore List in Settings. One list now covers chat, credits, and future tools. Your old chat ignore list moves over automatically.
Added - Credits tab now shows exactly who will appear in the credits, section by section, as viewers chat.
Added - Pomodoro can now be shown in the master overlay ("Also show in master overlay" checkbox).

Fixed - Channel point redeems firing twice (double song requests, double wheel spins).
Fixed - Song requests replaying already-played songs after the queue self-check kicked in.
Fixed - Wheel overlay landing on the wrong item and visually "jumping" to the real winner after a spin.
Fixed - Wheel overlay sometimes spinning twice from a single redeem.
Fixed - Credits showing "no activity" if the app was restarted after stream. Collected chatters now survive an app restart (up to 6 hours).
Fixed - Credits preview not updating when changing settings. It now updates live, using sample names until real chatters come in.
Fixed - Pomodoro timer not showing on its overlay.
Fixed - Timer overlay box growing/shrinking as the numbers changed. The box now holds its size.
